Команды Docker

  • docker ps - отображет запущенные контейнеры
  • docker stop - останавливает контейнер
  • docker run - создает новый контейнер и сразу включает его
  • docker pull - загружает образы из удаленного репозитория Docker Hub. Позволяет получить доступ к готовым образам.
  • docker rm - удаляет контейнер
  • docker restart - Перезапускает контейнер
