Git

  • git init - Инициализирует новый репозиторий Git.
  • git clone - Создает копию существующего репозитория Git.
  • git status - отображает состояние рабочего каталога, какие файлы отслеживаются.
  • git add - добавляет файл в отслеживаемые.
  • git commit - фиксирует изменения, внесенные в репозиторий, создавая новый коммит.
  • git push - используется для отправки локальных изменений в удаленный репозиторий.
  • git branch - используется для работы с ветками: создания, просмотра, переименования и удаления.
  • git checkout - позволяет перемещаться между ветками.
  • git pull - предназначена для получения изменений из удаленного репозитория и автоматического объединения их с локальной веткой.
